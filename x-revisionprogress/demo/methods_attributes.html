Attributes
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
range
'[begin,end]' This attribute could be filled using the event bus. (see period-context). It represents the range of the full BAR containing this progress.
period-context
This parameter identify the context for getting period refresh (if not defined, any range is received).
machine-id
Id of the machine.
revision-id
Id of the revision to track.
kind
Kind of revision to track.
revision-range
Range of the revision to track.
steps
Total number of steps. For a testing purpose.
remaining
Number of remaining steps. For a testing purpose.
