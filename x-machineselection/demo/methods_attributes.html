Attributes
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
unique-machine
Set this parameter to "true" so that only one machine can be selected in the dialog.
Note: when this option is set, dynamic groups cannot be displayed even with "enableGroups".
ordering
Set this parameter to "true" so that you can reorder the selected machines.
In that case, selecting a category will result in selecting all individual machines within this category with the right order.
enableGroups
Set this parameter to "true" if you want to display dynamic groups. By default this is false. Can be defined globally in config
Note: this option is deactivated if used together with "unique-machine".
machine
List of selected machine ids, each id being separated by a comma. -1 means "all machines".
group
List of selected group ids, for example "C4" or "Ca2,ET0".
