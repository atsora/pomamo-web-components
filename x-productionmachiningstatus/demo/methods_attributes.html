Attributes
machine-id
Id of the machine to display in this widget.
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
thresholdunitispart
Threshold unit. Default true. (false = percent)
thresholdorangeproduction
Threshold to display orange color. Default 0. Lower than next, else ignored.
thresholdredproduction
Threshold to display red color. Default 0. Greater than previous
display-context
Context to delegate "2/3" display to x-remotedisplay / x-production
bar-style
'true' to display like others BARS - 'false' to display text in a box.
