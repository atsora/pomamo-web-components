Attributes
machine-id
Id of the machine to display in this widget.
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
threshold1
This parameter sets a delay such as if next stop will happens after this delay, status circle has a normal state, under which status circle go to state warning.
threshold2
This parameter sets a delay such as if next stop will happens before this delay, status circle has a emergency state and status circle will blink.
If next stop delay is between threshold1 and threshold2, status circle take warning state.
