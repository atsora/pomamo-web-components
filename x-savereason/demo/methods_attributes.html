Attributes
machine-id
Id of the machine to display in this widget.
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
ranges
List of ranges to use at start (range&range2) replaces several calls to addPeriod.
Methods
addRange (range)
Add a period to display.
removeRange (range)
Remove a period to display.
cleanRanges ()
Remove all displayed periods.
