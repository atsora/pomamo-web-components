Attributes
machine-id
Id of the machine to display in this widget. This parameter can be used to complete motion-context when using x-grouparray.
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
range
'[begin,end]' This attribute could be filled using the event bus. (see period-context).
period-context
This parameter identify the context for getting period refresh (if not defined, any range is received).
machine-context
This context is used for receiving machine-id messages, for example when clicking on a tab.
