Attributes
range
'[begin,end]' This attribute could be filled using the event bus. (see period-context).
period-context
This parameter identify the context for getting period refresh (if not defined, any range is received).
height
Height of widget.
Methods
addRange (start, end)
Add a range to display.
removeRange (start,end)
Remove a range to display.
cleanRange ()
Remove all displayed ranges.
