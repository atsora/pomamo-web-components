Attributes
machine-id
Id of the machine to display in this widget. This parameter can be used to complete motion-context when using x-grouparray.
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
range
'[begin,end]' This attribute could be filled using the event bus. (see period-context).
height
Height of the widget
motion-context
This parameter identify the context for sending motion in hours and percent (if not defined, nothing is sent).
period-context
This parameter identify the context for getting period refresh (if not defined, any range is received).
machine-context
This context is used for receiving machine-id messages, for example when clicking on a tab.
showoverwriterequired
This parameter defines if a black line should be displayed or not.
click
Set the behavior of this widget in case of a click. Allowed values : change, details, popup, none.
showdetails and showpopup
Comma separated list of components to display in the details or popup page. To set only if click=details.
