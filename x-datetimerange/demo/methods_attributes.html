Attributes
range
'[begin,end]' in ISO 8601 date in string format : yyyy-MM-ddTHH:mm:ssZ
min-begin
Minimum value of begin attribute may take. Attribute value has following format: yyyy-MM-ddTHH:mm:ssZ (ISO 8601 date in string format)
max-begin
Maximum value of begin attribute may take. Attribute value has following format: yyyy-MM-ddTHH:mm:ssZ (ISO 8601 date in string format)
min-end
Minimum value of end attribute may take. Attribute value has following format: yyyy-MM-ddTHH:mm:ssZ (ISO 8601 date in string format)
max-end
Maximum value of end attribute may take. Attribute value has following format: yyyy-MM-ddTHH:mm:ssZ (ISO 8601 date in string format)
no-end
Tells if datetime range could not have an end value. Default value is false
period-context
This parameter identify the context for getting period refresh (if not defined, any range is received).
not-editable
True if datetime range can not be changed
hide-buttons
True hide buttons.
datetime-context
This parameter identify the context for getting small vertical red line position (= when ). Useful for zoom.
