Attributes
templateid
Id of the box to clone
group
This parameter is a group to read groups (= list of machines) to display. (ex: Ca1, Co2) (auto read from config).
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
machine
This parameter is used if groups is empty. It is a fixed list of machines to display. (ex: 1,2,3) (auto read from config)
column
Number of columns to display (auto read from config).
row
Number of rows to display (auto read from config).
canUseRowsToSetHeight
True if rows height should be forced using page height and number of rows. (auto read from config).
forcestaticlist
True if group must be loaded only 1 time and never reloaded.
donotwarngroupreload
True to avoid unwanted field legend reload (default false, could try to reload groups and machines everywhere).
