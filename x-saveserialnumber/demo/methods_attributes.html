Attributes
machine-id
Id of the machine to display in this widget.
path
This parameter can be used for tests. It is read in configuration filled by x-checkpath.
range
It contains the range of the serial number scanned. '[begin,end]'
serial-number
Serial number assigned to cycle at date/time of scan, if it exists.
datetime
Begin or End of the period
is-begin
Datetime is begin ?
