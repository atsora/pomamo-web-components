Attributes
period-context
This parameter identify the context for SENDING period refresh. If not defined, send to all.
displaydaysrange
This parameter defined the number of days to display. (default = 1).
displayhoursrange
This parameter defined the number of hours to display. (un-used if displaydaysrange or displayweekrange is used).
displayshiftrange
This parameter defines when shift should be displayed. WARNING ! Database should be configured accordingly, else current day will be used.
displayweekrange
This parameter defines when WEEK should be displayed. WARNING ! Database should be configured accordingly, else current day will be used. (un-used if displaydaysrange is used).
exclude-now
This parameter tells if the current period should be displayed or not. (default = display today)
textchange-context
Context to send a text to display (== shift name) EVEN if it contains the substring "machine-id", it WON'T be used to complete it.
